Redis常用命令

默认16个数据库.初始默认使用0号库. 所有库密码相同

select 切换数据库
keys *
exists key [key ...]
type key
del key [key ...]
expire key seconds
ttl key 查看剩余过期时间
dbsize 查看当前数据库key的数量
flushdb 清空当前库
flushall 清空全部库
get key
set key value
append key value
strlen key
setnx key value 只有key不存在时执行

incr key
incrby key increment
decr key
decrby key decrement

mset key value [key value ...]
mget key [key ...]
msetnx key value [key value ...]

getrange key start end
getrange key start -1 倒数第一个
setrange key offset value

setex key seconds value 设置键值同时设置过期时间
getset key value 设置新值同时获得旧值
